# 出差检查清单模板

按出差场景选择并合并到最终行程单。不要输出与本次出差明显无关的项目。

## 通用清单

### 证件类
- [ ] 身份证
- [ ] 名片
- [ ] 工牌/门禁卡（如需进入客户办公区）

### 设备类
- [ ] 笔记本电脑 + 充电器
- [ ] 手机 + 充电器
- [ ] 充电宝
- [ ] 耳机
- [ ] 转接头/扩展坞
- [ ] 翻页笔（如需演示）

### 资料类
- [ ] 客户背景资料
- [ ] 拜访议程
- [ ] 演示 PPT/原型/报价材料
- [ ] 合同/协议/盖章材料（如需签署）
- [ ] 项目交付物或会议纪要模板

### 预订确认
- [ ] 确认航班/高铁班次和票面时间
- [ ] 确认酒店门店、房型和入住日期
- [ ] 确认客户拜访时间、联系人和到访登记要求
- [ ] 在分贝通完成机票/高铁/酒店预订
- [ ] 保存电子发票/报销凭证

## 当天往返

- [ ] 确认返程前是否有足够时间从客户处到机场/车站
- [ ] 预留晚高峰或恶劣天气交通缓冲
- [ ] 确认随身行李可直接带入客舱/车厢

## 过夜出差

- [ ] 确认酒店最晚入住时间
- [ ] 确认酒店到首个拜访地点的通勤时间
- [ ] 准备换洗衣物和常用药品

## 多客户拜访

- [ ] 确认客户之间转场路线和预计耗时
- [ ] 给每场会议预留至少 15 分钟收尾和转场缓冲
- [ ] 按客户分别准备材料和联系人信息
